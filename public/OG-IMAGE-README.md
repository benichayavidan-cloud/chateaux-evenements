# Image Open Graph (og-image.jpg)

## ⚠️ IMPORTANT - Action requise

Le fichier `og-image.jpg` actuel est un **placeholder temporaire**.
Vous devez le remplacer par une **vraie image professionnelle** avant le go-live.

## Spécifications requises

- **Dimensions:** 1200x630 pixels (exact)
- **Format:** JPG ou PNG
- **Poids:** < 300 KB (recommandé)
- **Qualité:** Haute résolution

## Contenu recommandé

L'image doit contenir :

1. **Logo Select Châteaux** (centré en haut)
2. **Photo d'un château** (arrière-plan subtil)
3. **Texte:**
   - "Select Châteaux"
   - "4 Châteaux d'Exception en Île-de-France"
   - "www.selectchateaux.com"
4. **Couleurs:** Or (#a37e2c), gris élégant
5. **Style:** Luxe, épuré, professionnel

## Outils recommandés

- **Canva** (gratuit) - Template "Facebook Post" 1200x630
- **Figma** (gratuit)
- **Photoshop**
- **GIMP** (gratuit)

## Checklist avant go-live

- [ ] Remplacer og-image.jpg par image professionnelle
- [ ] Vérifier dimensions exactes (1200x630)
- [ ] Tester sur Facebook Debugger: https://developers.facebook.com/tools/debug/
- [ ] Tester sur Twitter Card Validator: https://cards-dev.twitter.com/validator
- [ ] Optimiser poids fichier (< 300 KB)

## Emplacement

Placer le fichier final ici :
`/public/og-image.jpg`

Il sera automatiquement utilisé sur toutes les pages du site.
