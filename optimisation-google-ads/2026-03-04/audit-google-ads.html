Plan d'action - Recommandations
URGENT

1. Vérifier le tracking des conversions

C'est la priorité absolue. Sans tracking, tout l'argent est jeté par la fenêtre.

  • Vérifier que le tag Google Ads est installé sur le site
  • Configurer une action de conversion : formulaire de devis envoyé
  • Tester avec le Tag Assistant de Google
  • Vérifier les Enhanced Conversions (déjà codées dans le site)
URGENT

2. Corriger la devise du compte

Ton compte est en ILS (Shekel). Pour un business français ciblant la France :

  • Vérifier le paramètre de devise dans Google Ads
  • Vérifier le ciblage géographique (France uniquement ?)
  • Note : la devise ne peut pas être changée après création du compte
URGENT

3. Nettoyer les mots-clés

Supprimer les 48 mots-clés "rarement diffusés" et optimiser :

  • Garder : [location chateau séminaire], [séminaire chantilly], [séminaire vallée de chevreuse]
  • Passer en large : les mots-clés géo (séminaire 78, 92, 60)
  • Supprimer : tous les "rarement diffusés" sans volume
  • Ajouter des négatifs : gratuit, pas cher, mariage, anniversaire
IMPORTANT

4. Optimiser les landing pages

Même si le CTR est bon, les pages doivent convertir :

  • CTA "Demander un devis" visible sans scroller
  • Formulaire court (nom, email, téléphone, date, nb personnes)
  • Numéro de téléphone cliquable en haut
  • Témoignages clients / logos entreprises
  • Vitesse de chargement < 3 secondes
IMPORTANT

5. Améliorer l'annonce Paris 92

Seule annonce notée "Moyenne" par Google :

  • Ajouter plus de mots-clés dans les titres
  • Ajouter plus de mots-clés dans les descriptions
  • Le CPC est le plus bas (17,23 ₪) = bon potentiel si amélioré
MOYEN TERME

6. Augmenter le budget intelligemment

Une fois le tracking fixé et les conversions mesurées :

  • Réallouer le budget vers les mots-clés qui convertissent
  • Augmenter SNIPER_GEO_IDF (actuellement limité)
  • Envisager Performance Max avec audience personnalisée
  • Tester des extensions d'annonces (lieu, appel, liens annexes)
